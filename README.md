# CodeSnippet Hub

CodeSnippet Hub - это настольное приложение для хранения и управления кодовыми сниппетами с подсветкой синтаксиса. Это ваша личная библиотека кода, где вы можете хранить полезные куски кода, которые часто используете, но забываете.

## Особенности

- Создание заметок с заголовком и тегами
- Подсветка синтаксиса для различных языков программирования
- Кнопка "Копировать в буфер" в один клик
- Поиск по заголовкам и тегам
- Поддержка многоячеистых сниппетов (код, текст, изображения)
- Локальное хранение данных в SQLite
- Режим изучения сниппетов
- Большой редактор для удобного редактирования
- Гибкое количество колонок в сетке сниппетов

## Технологии

- **Python** - основной язык программирования
- **Flet** - создание настольного интерфейса
- **SQLite** - локальная база данных
- **PyYAML** - сериализация многоячеистых сниппетов
- **pyperclip** - копирование в буфер обмена
- **uv** - менеджер зависимостей (альтернатива pip)

## Установка и запуск

### Требования

- Python 3.8.1 или выше
- [uv](https://github.com/astral-sh/uv) (рекомендуется) или pip

### Установка с помощью uv (рекомендуется)

1. Установите uv:
   ```bash
   pip install uv
   ```

2. Установите зависимости:
   ```bash
   uv sync
   ```

3. Запустите приложение как настольное:
   ```bash
   uv run python src/main.py
   ```

### Альтернативная установка с помощью pip

1. Клонируйте или скачайте этот репозиторий

2. Перейдите в директорию проекта:
   ```bash
   cd codesnippet-hub
   ```

3. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # На Linux/Mac
   # или
   venv\Scripts\activate  # На Windows
   ```

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустите приложение:
   ```bash
   python src/main.py
   ```

## Сборка в исполняемый файл

Для создания исполняемого файла используйте Flet pack:

```bash
pip install pyinstaller
flet pack src/main.py --name CodeSnippetHub
```

## Функциональность

### Основной экран

- Поиск сниппетов по названию, языку или тегам
- Кнопка добавления нового сниппета
- Сетка карточек со сниппетами с возможностью изменения количества колонок
- Кнопка перехода в режим большого редактора

### Карточка сниппета

- Название и основной язык программирования
- Область с подсветкой синтаксиса для многоячеистых сниппетов
- Кнопки: копировать, редактировать, удалить, изучать

### Редактор сниппетов

- Поддержка многоячеистых сниппетов (код, текст, изображения)
- Выбор типа ячейки (код, текст, изображение)
- Выбор языка программирования для ячеек кода
- Возможность добавления нескольких ячеек в одном сниппете
- Сохранение в YAML формате

### Режим изучения

- Специальный режим для изучения сниппетов
- Удобное отображение содержимого сниппета
- Кнопка возврата к основному экрану

### Большой редактор

- Расширенный режим редактирования сниппетов
- Более удобный интерфейс для работы с многоячеистыми сниппетами
- Возможность создания новых сниппетов в расширенном режиме

## Архитектура

Проект использует архитектурный подход, разделенный на следующие слои:

- **UI слой** (src/ui): компоненты интерфейса, диалоги, представления
- **Модель слой** (src/models): работа с базой данных, хранение данных
- **Основной слой** (src/main.py): координация между UI и моделью

## Хранение данных

- Данные хранятся в локальной SQLite базе данных
- Сниппеты хранятся в формате YAML для поддержки многоячеистой структуры
- Поддержка тегов для категоризации сниппетов
- Поиск по заголовкам, языкам и тегам
